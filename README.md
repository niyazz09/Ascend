# About

ASCEND is a next-generation AI-powered learning platform designed to deliver personalized educational experiences through the collaboration of multiple specialized AI agents.

Unlike traditional learning systems that provide identical content to every learner, ASCEND dynamically adapts learning paths according to individual learning needs, knowledge level, and progress.

By orchestrating multiple AI agents, the platform automatically generates personalized roadmaps, detailed explanations, quizzes, and mastery assessments from a single user prompt, making learning more efficient, engaging, and outcome-driven.

---

# Problem Statement

Traditional educational platforms often rely on static content and require educators to use multiple disconnected tools for curriculum design, content creation, assessments, and learner tracking. This process is time-consuming and difficult to personalize.

ASCEND solves this challenge by introducing a Multi-Agent AI Platform where specialized AI agents collaborate intelligently to create complete personalized learning experiences from a single prompt. The platform automates planning, content generation, assessment, and progress tracking while adapting to each learner's needs.

# 🧠 AI Agents

## 📅 Planner Agent

Creates a personalized study plan based on the learner's goals, experience level, and available study time.

---

## 📚 Theory Agent

Generates easy-to-understand explanations with structured learning content, examples, and concept breakdowns.

---

## 🛣 Roadmap Agent

Builds a step-by-step learning roadmap that guides learners from beginner to advanced levels.

---

## 📝 Quiz Agent

Creates adaptive quizzes that test conceptual understanding rather than memorization.

---

## 📊 Assessment Agent

Evaluates learner performance, calculates mastery levels, and provides personalized feedback for continuous improvement.

# ⚙ Installation

## Clone Repository

```bash
git clone  https://github.com/niyazz09/Ascend.git
```

---

## Frontend

``bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

npm install

npm start
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

GEMINI_API_KEY=your_api_key

JWT_SECRET=your_secret

DATABASE_URL=your_database_url
```
